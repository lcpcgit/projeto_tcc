projeto tcc sobre sistema de predicao de vendas de componemtes hardware



Módulo 1: Acesso e Segurança

Autenticação de Usuário: Tela de login para proteger dados financeiros e estratégicos da empresa


Módulo 2: Robô de Captura de Mercado (Web Scraping)

Coleta Automatizada: Script rodando em segundo plano que acessa (ex: Kabum, Terabyte) para capturar o preço atualizado de hardwares específicos

Armazenamento de Histórico Externo: Gravação semanal dessa flutuação de preços globais no banco de dados para consulta da IA


Módulo 3: Ingestão, Tratamento e Exportação de Dados

Data Cleaning: Upload da base de vendas internas da empresa com um sistema que limpa dados nulos, remove duplicatas e padroniza nomes de componentes e etc tudo automaticamente

Exportação Flexível: Opção para baixar a base de dados tratada em formato Excel ou CSV


Módulo 4: Dashboard de Visão Estratégica

Gráficos Interativos: Painel visual onde o usuário pode explorar os dados

Visão Comparativa (Interno vs. Externo): Gráficos de linha que sobrepõem o preço da empresa com o preço do  mercado em geral ao longo do tempo 


Módulo 5: Motor de Previsão com Inteligência Artificial

Previsão de Demanda: O Machine Learning analisa o histórico interno e os dados do mercado para prever o volume de vendas futuras de um hardware

Intervalos de Confiança: O sistema fornece faixa segura (ex: "entre 140 e 160")
